[A0] - Impressão do Relatório Basic Path
<p>1. O SIstema apresenta a página de confirmação do relatório (o mesmo não tem filtro);<br/></p><p>2. O Ator clica no botão [OK]; [A1]<br/></p><p>3. O Sistema pesquisa as informações e apresenta a informações na tela no formato de PDF;<br/></p><p>4. O Ator opta por imprimir ou fechar a Janela;<br/></p><p>5. O Caso de Uso é encerrado<br/></p>
[A1] - Não existe informações para impressão Alternate
<p>1. O Sistema apresenta uma popup com a mensagem de informações insuficientes para gerar o relatório requisitado;<br/></p><p>2. O Ator clica no "OK", fechando a janela;<br/></p><p>3. O Sistema retorna ao passo 2 do fluxo principal<br/></p>
