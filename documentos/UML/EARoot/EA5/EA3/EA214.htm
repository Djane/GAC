Scenario Type Detail
.Manutenção dos Contratos Basic Path
<p>1. O Ator escolhe o item "Contratos" do Menu Principal.<br/></p><p>2. O Sistema apresenta a página de pesquisa de Contratos<br/></p><p>3.  Ator preenche qualquer campo para efetuar a pesquisa.[A1][A2][A3]<br/></p><p>4. O Sistema preenche a grid abaixo com o resultado da busca efetuada pelo ator. [A4][A5][A6]<br/></p><p>5. Caso de uso é encerrado<br/></p>
[A1] - Ator clica no botão "Fechar" Alternate
<p>1. O Ator clica no botão "Fechar" em qualquer ocasião<br/></p><p>2. O Sistema fecha a página e retorna a página principal.<br/></p>
[A2] - Ator clica no botão "Limpar" Alternate
<p>1. O Ator em qualquer situação clica no botão "Limpar"<br/></p><p>2. O Sistema limpa os dados digitados nos campos de filtro<br/></p><p>3. O fluxo retorna ao passo 2 do Fluso Principal<br/></p>
[A3] - Ator clica no botão "Novo" Alternate
<p>1. O Ator a qualquer momento, clica no botão "Novo"<br/></p><p>2. O Sistema abre a página de cadastramento de contrato<br/></p><p>3. O Ator preenche as informações relativas ao contrato<br/></p><p>4. O Sistema grava as informações e ativa a aba de "Cliente"<br/></p><p>5. O Ator preenche as informações relativas ao Cliente<br/></p><p>6. O Sistema grava as informações relativas ao Cliente e ativa a aba de "Dispositivos"<br/></p><p>7. O Ator associa os dispositivos ao Cliente e as suas respectivas Centrais<br/></p><p>8. O Sistema grava as informações relativas aos dispositivos e ativa a aba "Doenças"<br/></p><p>9. O Ator associa as doenças ao Cliente<br/></p><p>10. O Sistema grava as informações de doenças e ativa a aba "Tratamento"<br/></p><p>11. O Ator preenche as informações relativas ao tratamento ou medicação ao Cleinte<br/></p><p>12. O Sistema grava as informações relativas a tratamento do Cliente e ativa a aba "Contato"<br/></p><p>13. O Ator preenche a Aba de Contato e clica em "Salvar". [A4][A5]<br/></p><p>14. O Sistema efetiva a gravação de todas as informações no banco de dados e retorna ao passo 1 do Fluxo principal<br/></p>
[A4] - Ator clica no botão "Formas de Contato" Alternate
<p>1. O Ator na aba de "Contatos" clica no botão de "Formas de Contato"<br/></p><p>2. O Sistema grava o contato no banco e abre um popup de informações de "Formas de Contato"<br/></p><p>3. O Ator escolhe a forma de contato e preenche as informações da mesma.<br/></p><p>4. O Sistema efetiva a gravação da forma de contato e retorna ao passo 3. [A6]<br/></p><p>5. O Ator clica no botão "Fechar"<br/></p><p>6. O Sistema volta ao passo 12 do fluxo alternativo [A3].<br/></p>
[A5] - Ator clica no botão "Adicionar" da aba "Contatos" Alternate
<p>1. O ator dentro da aba "Contatos" clica no botão "Adicionar"<br/></p><p>2. O sistema limpa todos os campos de edição e leva o cursor para o primeiro campo e desvia seu fluxo para o passo 12 do fluxo alternativo [A3]<br/></p>
[A6] - Ator clica no botão "Adicionar" do popup "Formas de Contato" Alternate
<p>1. O ator dentro do popup "Forma de Contatos" clica no botão "Adicionar"<br/></p><p>2. O sistema limpa todos os campos de edição e leva o cursor para o primeiro campo e desvia seu fluxo para o passo 3 do fluxo alternativo [A4]<br/></p>
