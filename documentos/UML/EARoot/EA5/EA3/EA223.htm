Scenario Type Detail
- CRUD de SMS Padrão Basic Path
<p>1. O Ator clica no item do submenu "SMS Padrão", do menu "Cadastros";<br/></p><p>2. O Sistema apresenta a página de "Manutenção de SMS Padrão";<br/></p><p>3. O Ator digita as informações do usuário e clica em Salvar; [A1], [A2]<br/></p><p>4. O Sistema efetiva a gravação e apresenta um popup do status da gravação;<br/></p><p>5. O Ator confirma finalização de tarefa;<br/></p><p>6. Caso de Uso é encerrado, volta ao menu principal.<br/></p>
[A1] - Alteração das Informações Alternate
<p>1. O Ator clica no botão de edição das informações;<br/></p><p>2. O Sistema efetiva a leitura das informações no banco e carrega a página de edição das informações;<br/></p><p>3. O Ator efetua as alterações necessárias e clica no botão salvar;<br/></p><p>4. O Sistema efetiva a gravação e apresenta um popup do status da gravação;<br/></p><p>5. O Ator confirma a efetivação.<br/></p>
[A2] - Exclusão de Informações Alternate
<p>1. O Ator clica no botão de exclusão das informações;<br/></p><p>2. O Sistema apresenta um popup de confirmação de exclusão;<br/></p><p>3. O Ator confirma a exclusão;<br/></p><p>4. O Sistema efetiva a exclusão das informações e apresenta um popup de efetivação da transação;<br/></p><p>5. O Ator confirma a efetivação.<br/></p>
