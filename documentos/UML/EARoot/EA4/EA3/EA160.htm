Scenario Type Detail
- Login da Aplicação Basic Path
<p>1. O Ator digita a URL de acesso a Aplicação;<br/></p><p>2. O Sistema apresenta a tela de Login;<br/></p><p>3. O Ator digita o Login e a Senha e no botão de entrada do Sistema; [A2], [A3], [A4]<br/></p><p>4. O Sistema direciona para opção escolhida pelo Ator; [A5], [A6]<br/></p><p>5. Caso de uso encerrado;<br/></p><p>6. Caso de Uso é encerrado, volta ao menu principal.<br/></p>
[A2] - Ator deixa o Login em branco Alternate
<p>1. O Sistema apresenta um popup de "Falta de digitação do Login";<br/></p><p>2. O Ator clica no botão "OK" para fechar a popup;<br/></p><p>3. O Sistema é redirecionado para o passo 2 do caminho principal.<br/></p>
[A3] - Ator deixa a senha em branco Alternate
<p>1. O Sistema apresenta um popup de "Falta de digitação da Senha";<br/></p><p>2. O Ator clica no botão "OK" para fechar a popup;<br/></p><p>3. O Sistema é redirecionado para o passo 2 do caminho principal.<br/></p>
[A4] - Login ou senha estão erradas Alternate
<p>1. O Sistema apresenta um popup de "Erro de digitação do Login ou senha";<br/></p><p>2. O Ator clica no botão "OK" para fechar a popup;<br/></p><p>3. O Sistema é redirecionado para o passo 2 do caminho principal.<br/></p>
[A5] - Clique Menu Principal Alternate
<p>1. O Ator clicou no botão "Menu Principal";<br/></p><p>2. O Sistema é redirecionado para o "Menu Principal";<br/></p><p>3. Caso de uso encerrado.<br/></p>
[A6] - Clique na página de Ocorrências Alternate
<p>1. O Ator clicou no botão "Ocorrência";<br/></p><p>2. O Sistema é redirecionado para a página "Ocorrências";<br/></p><p>3. Caso de uso encerrado.<br/></p>
