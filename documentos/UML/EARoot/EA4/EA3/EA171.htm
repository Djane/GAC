Scenario Type Detail
- Carga de Lote de Dispositivos Basic Path
<p>1. O Ator clica o item de menu "Carga de Dispositivos" do menu principal;<br/></p><p>2. O Sistema apresenta a página de carga de dispositivos, via arquivo texto;<br/></p><p>3. O Ator escolhe o arquivo texto que contém as informações para carga de dispositivos e clica no "OK";<br/></p><p>4. O Sistema efetua a carga das informações e apresenta uma página com as criticas da carga ou um  popup de confirmação; [A1]<br/></p><p>5. O ator clica no "OK";<br/></p><p>6. Caso de Uso é encerrado, volta ao menu principal.<br/></p>
[A1] - Falha na carga de dispositivos Alternate
<p>1. O Sistema apresenta uma página com a lista de incompatibilidades geradas na carga;<br/></p><p>2. O Ator poderá ler a lista ou optar por gerar um "PDF" com os problemas encontrados;<br/></p><p>3. O caso de uso retorna ao passo 2 do fluxo principal.<br/></p>
